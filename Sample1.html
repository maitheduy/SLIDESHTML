Bài 1: Tích vô hướng

Vector A = ⟨a₁, a₂, a₃⟩ Vectơ A = ⟨a₁, a₂, a₃⟩
Length: |A| = √(a₁² + a₂² + a₃²) Độ dài: |A| = √(a₁² + a₂² + a₃²)

⚫ Tích vô hướng

A·B = a₁b₁ + a₂b₂ + a₃b₃ A·B = a₁b₁ + a₂b₂ + a₃b₃
A·B = |A||B|cos θ A·B = |A||B|cos θ
